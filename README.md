# CashBhak
Sebuah Web yang berfungsi sebagai kasir, Web ini bisa menyimpan dan mendata produk yang anda jual. Selain itu akan ada analisis penjualan seperti grafik, piechart bahkan history penjualan.

## Fitur-fitur:
- Login Dan Register Yang berfungsi (Jika Menggunakan Database, dikarenakan database yang tidak bisa terpakai oleh vercel maka website akan static)
- Menambahkan produk ke database (Nama,Stok,Gambar)
- Fitur Transaksi yang akan menyimpan transaksi ke database
- Fitur dashboard yang akan diisi dengan hasil analisis transaksi transaksi yang anda lakukan di website

## Halaman Website:
- Landing Page
- Login
- Register
- Dashboard
- Transaksi
- Items
- Profile
